# Docker Exercises

This repository contains hands-on Docker exercises for Level Up In Tech students.

## Structure

The exercises are organized by week and session:

### Week 1 - Session 2: Docker Fundamentals
- Lab 1: Nginx Web Server
- Lab 2: Inspect Containers
- Lab 3: Container Logs
- Lab 4: Execute Commands
- Lab 5: Container Lifecycle
- Lab 6: Volume Mounts
- Lab 7: Multiple Containers
- Lab 8: Container Stats
- Lab 9: Cleanup

### Week 2 - Session 3: Building Docker Images
- Lab 1: First Dockerfile
- Lab 2: Layer Caching
- Lab 3: Base Images
- Lab 4: Optimization
- Lab 5: Multistage Builds

### Week 2 - Session 4: Volumes and Networking
- Lab 1: Volume Persistence
- Lab 2: MySQL Persistence
- Lab 3: Network DNS
- Lab 4: Multi-Container App
- Lab 5: Environment Configuration
- Lab 6: Backup and Restore

### Week 3 - Session 5: Docker Compose
- Lab 1: First Compose
- Lab 2: WordPress and MySQL
- Lab 3: Environment Configuration

### Week 3 - Session 6: Docker Swarm
- Lab 1: Swarm Basics
- Lab 2: Health Checks
- Lab 3: Stack Deployment

## Getting Started

Each lab folder contains its own README with detailed instructions. Navigate to the specific lab directory and follow the instructions provided.

## Prerequisites

- Docker installed on your system
- Basic command line knowledge
- Text editor of your choice

## Support

For questions or issues, please contact your Level Up In Tech instructor.
